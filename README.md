# time-series-analysis-R
Questo progetto contiene l’analisi esplorativa e statistica del dataset_08, 
utilizzando R. L’obiettivo è studiare le relazioni tra le variabili e 
produrre grafici e modelli utili all’interpretazione dei dati.

**Contenuto degli script**:

- pulizia dei dati
- analisi descrittiva
- visualizzazioni
- modelli statistici
